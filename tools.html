Tools

Some useful tools.

  • ROT13 Converter - does what it says on the tin.
  • GUID Generator - generate some random Globally Unique Identifiers.
  • MD5 and SHA Generator - generate md5, sha1 and sha256 values of the input string
  • Base 64 Encode - Encode and decode base 64 strings.
  • Lorem Ipsum Generator - Generates random lorem ipsum paragraphs.
  • Random Password Generator - Generate random easy to remember passwords.
  • ASCII Table - Simple list of ASCII values.
