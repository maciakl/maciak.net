About

This tool will generate some random easy to remember semi-pronounceable passwords.

Passes are generated by alternating consonants, vowels and numbers. Each password is broken into dash-separated syllables which are up to 4 characters long.

Default is 10 passwords, each 8 characters long. Max is 100 for both values.

Generator

If you are ok with the defaults, just hit Generate.

Passwords are dash-separated into easy to remember syllables.

You can keep these dashes as part of your password or drop them when you type it in. Dashes don't count against the character limit.
